## 10. 화면 전환 시나리오

### 10.1 주요 사용자 시나리오

```
시나리오 1: 아침 루틴 완료
─────────────────────────────────────────────────────
[홈] → [루틴 카드 체크] → [완료 모달] → [홈 업데이트]
                              │
                              └─ 애니메이션: 체크 + 축하 이펙트

시나리오 2: AI 상담 후 루틴 추가
─────────────────────────────────────────────────────
[코칭 탭] → [새 대화] → [대화 진행] → [리포트 생성]
                                          │
                                          ▼
                                    [리포트 화면]
                                          │
                                          ▼ "루틴 추가" 버튼
                                    [루틴 자동 생성]
                                          │
                                          ▼
                                    [루틴 탭으로 이동]

시나리오 3: 그룹 초대 받기 (Deep Link)
─────────────────────────────────────────────────────
[외부 링크 클릭] → [앱 실행/스플래시]
                        │
                        ├─ 로그인 됨: [그룹 가입 확인 모달] → [그룹 상세]
                        │
                        └─ 비로그인: [로그인] → [그룹 가입 확인] → [그룹 상세]

시나리오 4: 오프라인 → 온라인 전환
─────────────────────────────────────────────────────
[오프라인 상태에서 루틴 완료]
        │
        ▼
[로컬에 저장 (Pending)]
        │
        ▼ 네트워크 복구
[자동 동기화 시작]
        │
        ▼
[동기화 완료 토스트 표시]
```

### 10.2 애니메이션 타이밍

| 동작           | Duration | Curve       | 설명                     |
| -------------- | -------- | ----------- | ------------------------ |
| GNB 탭 전환    | 200ms    | ease-out    | Fade 전환                |
| 상세 화면 Push | 300ms    | ease-in-out | Slide from right         |
| 모달 등장      | 400ms    | ease-out    | Slide from bottom + Fade |
| 모달 닫힘      | 300ms    | ease-in     | Slide to bottom + Fade   |
| 체크박스 완료  | 300ms    | spring      | Scale + Check 애니메이션 |
| 카드 삭제      | 250ms    | ease-in     | Slide out + Fade         |
| 풀투리프레시   | 500ms    | ease-out    | Scale + Rotation         |
| 토스트         | 3000ms   | -           | 자동 dismiss             |
